# Side-Projects-and-Scripts-Public

A collection of side projects and utility scripts for various purposes.

## Projects

### [vaarattu-community-playlist-maker](vaarattu-community-playlist-maker/)

A Python script that creates Spotify playlists from Twitch channel points redemptions. Fetches song requests submitted through a Twitch channel points reward and compiles them into a Spotify playlist, featuring OAuth 2.0 authentication for both Twitch and Spotify APIs.
